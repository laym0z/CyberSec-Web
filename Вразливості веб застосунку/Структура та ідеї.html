Логін форма

1. Аутентифікація та авторизація

Функціонал:

  • Login / Register

  • Ролі: user / admin

  • Reset password

  • session fixation

Вразливості:

  • Weak password policy
  • Password bruteforce
  • Username enumeration (давати занадто багато інформації про статус логіну)✅
  • Broken access control (/admin без перевірки ролі)
  • Insecure session cookies (no HttpOnly / Secure)
  • Missing rate limiting (bruteforce)
  • SQL Injection
  • HTTP login
  • Вразливість відновлення паролю
  • Редірект на стороні клієнта

Брутфорсинг користувача (прибрати інформацію про користувача)✅ Брутфорсинг паролю (встановити обмеження по запитам) (Декілька даних за один запит) - Встановити перевірку на якість паролю - Дефолтні дані для логіну - SQL Inj - Маніпуляція з відповіддю сервера - Single Sign On - 2FA - Reset Password

2. Робота з формами

Функціонал:

  • Feedback form

  • Profile edit

Вразливості:

  • Stored XSS

  • Reflected XSS

  • Missing CSRF protection

  • HTML injection

3. Пошук і фільтрація

Функціонал:

  • Search users/products

Вразливості:

  • SQL Injection (classic + blind)

  • Error-based information disclosure

4. Завантаження файлів

Функціонал:

  • Upload avatar / document

Вразливості:

  • No MIME validation

  • Executable file upload

  • Path traversal (../)

  • Missing size limits (DoS vector)

5. API endpoints

Функціонал:

  • /api/users

  • /api/profile

Вразливості:

  • IDOR (Insecure Direct Object Reference)

  • Missing auth on API

  • Overexposed JSON fields

6. Конфігурація та інфраструктура

Функціонал:

  • .env

  • Debug mode

Вразливості:

  • Flask debug=True

  • Exposed stack traces

  • Hardcoded secrets

  • Misconfigured CORS

  • Open ports (DB, admin panel)

7. Логування і моніторинг (для SOC-демо)

Функціонал:

  • Access logs

  • Auth logs

Антипатерни:

  • Логи без ротації

  • Логи з паролями

  • Відсутність alerting
